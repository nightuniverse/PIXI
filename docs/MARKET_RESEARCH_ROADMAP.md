# 시장 조사 엔진 강화 로드맵

## 현재 상태
- ✅ 기본 시맨틱 검색 구현
- ✅ GPT 기반 키워드 확장
- ✅ 시장 규모 추정 (GPT 기반)
- ⏳ 실시간 웹 스크래핑 (준비됨, 미구현)
- ⏳ 외부 API 연동 (준비됨, 미구현)

## 우선순위별 구현 계획

### 🚀 Phase 1: 즉시 효과 (1-2일)
**목표: 실시간 경쟁사 정보 수집**

1. **네이버 검색 API 연동** (우선순위: 최상)
   - 네이버 검색 API로 경쟁사 실시간 검색
   - 검색 결과에서 스타트업 정보 추출
   - 장점: 빠른 구현, 무료 티어 제공, 한국 시장 특화
   - 예상 효과: 경쟁사 발견률 50% 향상

2. **앱스토어 크롤링** (우선순위: 높음)
   - Google Play Store 크롤링
   - 앱 이름, 다운로드 수, 리뷰, 평점 수집
   - 장점: 모바일 앱 경쟁사 정보 확보
   - 예상 효과: 모바일 앱 시장 분석 정확도 70% 향상

### 🔥 Phase 2: 중기 효과 (3-5일)
**목표: 시장 규모 및 통계 데이터 수집**

3. **통계청 Open API 연동** (우선순위: 높음)
   - 한국 표준산업분류(KSIC) 기반 시장 규모 조회
   - 산업별 매출, 성장률 데이터
   - 장점: 공식 통계 데이터, 신뢰도 높음
   - 예상 효과: 시장 규모 추정 정확도 80% 향상

4. **구글 검색 API 연동** (우선순위: 중간)
   - Google Custom Search API
   - 글로벌 경쟁사 정보 수집
   - 장점: 광범위한 검색 범위
   - 단점: API 비용 발생 가능

### 💎 Phase 3: 장기 효과 (1-2주)
**목표: 고급 분석 및 데이터베이스 구축**

5. **벡터 임베딩 기반 검색** (우선순위: 중간)
   - OpenAI Embeddings API 활용
   - 시맨틱 유사도 기반 경쟁사 검색
   - 장점: 키워드 매칭보다 정확한 검색
   - 예상 효과: 관련 경쟁사 발견률 90% 향상

6. **데이터베이스 구축** (우선순위: 중간)
   - PostgreSQL/MongoDB로 구조화된 데이터 저장
   - 인덱싱 및 캐싱으로 성능 향상
   - 장점: 빠른 검색, 데이터 누적
   - 예상 효과: 검색 속도 10배 향상

7. **크로스 검증 시스템** (우선순위: 낮음)
   - 여러 소스의 데이터 비교
   - 신뢰도 점수 계산
   - 장점: 데이터 품질 향상
